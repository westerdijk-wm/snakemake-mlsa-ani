Genome analysis pipeline report
================================

This report summarizes a modular comparative genomics workflow including:

- Genome quality control (QUAST)
- Gene extraction and quality control
- Multiple sequence alignment and concatenation (MLSA)
- Phylogenetic reconstruction (IQ-TREE, RAxML, or FastTree)
- Average Nucleotide Identity (ANI; skani, FastANI, or pyani)

All outputs are automatically generated by Snakemake and linked below.
